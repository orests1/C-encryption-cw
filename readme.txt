Call makefile to compile and execute program for all 
four given test files.

When compiling please use the -w command.

Program commands:

-e: Encryption Mode.
-d: Decryption Mode.
-i: Indicates that the next command is the input filename.
-o: Indicates that the next command is the output filename.
-a: Indicates that the next command is the encryption algorithm.
	(basic or xor)
-k: Indicates that the next command is the encryption key.

Commands work in any order.
